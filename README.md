## 🚀 Projekt Attekintes
Ez a projekt egy **termelesi szintu (production-grade)** mesterseges intelligencia ugynok-keretrendszer, amely a komplex vallalati automatizacios igenyek kielegitesere keszult. A megoldas az end-to-end AI fejlesztési folyamatokra fokuszal.

## 🛠️ Technikai Stack
| Komponens | Technologia |
| :--- | :--- |
| **Core AI** | Llama 3.1 / Groq API |
| **Kontenerizacio** | Docker |
| **Nyelv** | Python 3.11+ |
| **Monitorozas** | Rich CLI & Logger |

## 🏗️ Fobb Funkciok
* **End-to-End Architektura:** Koncepciotol a megvalositasig tervezett folyamatok.
* **RAG-Ready:** Modularis felepites a tudasbazisok hatékony integraciojahoz.
* **Hibaturlo:** Beepitett kivetelkezeles es naplozas az eles uzemelteteshez.
